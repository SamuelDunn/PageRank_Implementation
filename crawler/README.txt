Code Source: https://math.nist.gov/~RPozo/ngraph/webcrawler.html

The modular_crawler was run on 06/04/2020 to follow hyperlinks starting at 
https://www.ucsb.edu. 50 pages total were explored and all valid hyperlink associated 
with those pages were recorded. The output data from the crawl is in crawl_1_50.txt

The following command was used to execute the crawler code. 
$ python2 modular_crawler.py ucsb.edu http://www.ucsb.edu 50
